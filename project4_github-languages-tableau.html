Simple Dashboard with Tableau

In this project, I made a superficial analysis of Github programming languages dataset.

There are two datasets, one containing the number of repositories for each language and the other containing the number of issues recorded in each language from 2011 to 2022, separated by quarters. Both datasets can be found here.

Using R language and the RStudio software, a simple upload of the .csv files was made and then I described some information. Finally, a dashboard was generated in Tableau software.

Simple analysis

Two aggregations were made to find out which languages are most popular and have the highest number of issues.

This first one is to view the most popular languages on Github:

            repos_summary <- repos %>% group_by(language) %>% 
            arrange(desc(num_repos)) %>% head(15)
            View(repos_summary)

The result is:



And the other is the number of issues of each language:


            issues_summary <- issues %>% group_by(name) %>% 
            summarise(sum_issues = sum(count)) %>%
            arrange(desc(sum_issues)) %>% head(15)
            View(issues_summary)

The result is the following:



As we can see, Javascript is the most popular programming language on Github.

Dashboard

The dashboard was built using Tableau Desktop software.
